Excel导入家谱

📋 导入格式说明

Excel文件包含两个工作表：

  1. 第一个Sheet：家谱基本信息（只需填写一行）
  2. 第二个Sheet：人物信息，一行一个人

⬇️ 下载Excel模板

拖拽Excel文件到这里，或


已选择文件：

📝 字段说明（人物信息表）

表头 必填 说明
姓名* 是 人物姓名
字 否 古人的字
号 否 古人的号
性别 否 男/女
出生年份 否 例如：1800
逝世年份 否 例如：1880
父亲姓名 否 必须和Excel中父亲的姓名一致，用于建立关系
母亲姓名 否 必须和Excel中母亲的姓名一致
配偶姓名 否 必须和Excel中配偶的姓名一致
支系 否 例如：长房、二房
世代代数 否 从始祖开始算，始祖为1
排序序号 否 同辈中的排序，用于世系图排序
迁徙路线 否 描述迁徙过程
功绩/称号 否 主要成就或称号
传记 否 人物生平简介
传记来源 否 例如：族谱记载、墓碑
墓地位置 否 安葬地点
封号/爵位 否
官职/职业 否
婚姻状况 否 已婚/未婚等
状态 否 在世/去世
